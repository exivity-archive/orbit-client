# Introduction

Orbit-client is a declarative React client for Orbitjs.

Orbit is a framework for orchestrating access, transformation, and synchronization between data sources.
Visit https://orbitjs.com/ for more information.

Orbit-client enables you to request, transform and render your data effortlessly.
